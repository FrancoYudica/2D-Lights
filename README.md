
# 2D Lights

It's an offline 2D Light renderer. Uses a mix of Ray Marching and Path tracing techniques to render.

## Features

- Multiple primitives
- Materials:
    - Emission
    - Intensity
    - Reflectivity

- Path tracing:
    - Reflections
    - Refractions

- Outputs:
    -  .jpg image
    -  Frame sequences, can generate a sequence of images that when joined make a video
